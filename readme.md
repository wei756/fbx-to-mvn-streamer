# fbx-to-mvn-streamer

FBX 애니메이션을 Xsens MVN real-time network streaming Protocol을 이용해 스트리밍할 수 있는 프로그램입니다.

Xsens Analyze Plus/Pro 라이센스 없이 MVN 관련 장비 연동 소프트웨어(Unity 등) 개발에 이용할 수 있습니다.

### 지원되는 것

- 23개의 휴먼 본 데이터 전송

- 40개의 손가락 본 데이터 전송

- 타임 코드 데이터 전송

### 지원되지 않는 것

- 4개의 프롭 데이터 전송

- character id를 이용한 여러 개의 모션 동시 전송
